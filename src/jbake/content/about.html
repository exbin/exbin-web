title=About ExBin Project date=2016-08-09 type=page status=published ~~~~~~

ExBin Project is non-commercial initiative to create binary data centered communication protocol and family of file formats.

Main focus is given on abstraction, to specify meaning of data and to provide flexibility to combine various data types together and transform them when needed.

Advantages

  • Unified Data Structures - Unified tree structure should allow to combine any types of data
  • Compatibility - Framework should provide data transformations/processing capability to solve compatibility issues
  • Comprehensibility - Catalog of data types, metadata, relations and abstraction should allow better understanding of data
  • Efficiency - Support for compression and encryption on sections or whole document should allow more secure and more effective representation of binary data

Project Scope

Range of libraries, parsers, utilities and application based on XBUP protocol are part of the project.

  • Protocol specification
  • Libraries for parsing and serialization to/from protocol
  • Editor for files encoded with protocol
  • Shared catalog of data types
  • Application service layer provider

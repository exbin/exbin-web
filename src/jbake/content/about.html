title=About ExBin Project date=2017-01-23 type=page status=published ~~~~~~

ExBin project is an initiative to bring evolution in binary data storing and processing.

Main focus is provide new means of working with binary data providing:

  • Universal Usage - Allow to combine various data types in forms most suited for the specific needs
  • More Abstraction - Ability to declare and specify meaning of data with desired level of depth
  • Flexibility - Support for transformation operations and service for data processing and compatibility handling

Project is currently in early alpha stage with partial prototype implementation.

Project Scope

  • eXtensible Binary Universal Protocol [XBUP]
  • Catalog of Data Types
  • Transformation framework
  • Programming Language [ProgJazy]
  • Examples and demonstration applications
